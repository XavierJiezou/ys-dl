## Install

```bash
pip install ys-dl
```

## Usage

`$ ys-dl --help`

- icon

```bash
ys-dl icon # Downlaod 256x256 icon by default
ys-dl icon --size 32 # Download 32x32 icon
```

- rank

```bash
ys-dl icon # Downlaod 256x256 logo by default
ys-dl icon --size 32 # Download 32x32 logo
```

- cos

```bash
ys-dl icon # Downlaod 256x256 logo by default
ys-dl icon --size 32 # Download 32x32 logo
```

- fan

```bash
ys-dl icon # Downlaod 256x256 logo by default
ys-dl icon --size 32 # Download 32x32 logo
```

- topic

```bash
ys-dl icon # Downlaod 256x256 logo by default
ys-dl icon --size 32 # Download 32x32 logo
```
